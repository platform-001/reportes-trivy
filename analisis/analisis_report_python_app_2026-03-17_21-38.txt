1. CVE-2023-1255 | libssl3 | CRITICAL | OpenSSL buffer overread in AES-XTS cipher

Description:
This vulnerability is a buffer overflow issue in the implementation of the AES-EXTSIV (AES Extended Short Insertion Valve) algorithm in OpenSSL 1.1.1 and earlier. It affects the handling of certain inputs by this algorithm, which could cause it to dereference or manipulate memory beyond its expected range.

Impact:
This vulnerability could lead to a DoS (Denial of Service) attack, as it allows an attacker to send a maliciously crafted message that causes the server to crash or become unresponsive, potentially resulting in data loss or unauthorized access.

Solutions:
To mitigate this vulnerability, follow these steps:
1. Check if your system is affected by this vulnerability and patch accordingly.
2. Implement a buffer overflow check for the AES-EXTSIV algorithm in your codebase.
3. Update your OpenSSL library to the latest version (which may already include this fix).
