REPORTE DE SEGURIDAD
====================
Artefacto: mysql:8.0.32
Fecha: 16/03/2026 19:09
====================

Resumen: 

- CVE-2023-21980 es un vulnerabilidad de seguridad en la versión de MySQL Server (mysql-server) que se encuentra en desarrollo. Se presenta como un ataque de privilegios desde el núcleo de servidor, permitiendo el acceso a datos críticos en una base de datos.
- CVE-2023-21977 es un problema de seguridad al nivel del núcleo de Servicios de base de datos (dbsvcs), que permite el acceso a datos vulnerables desde el controlador de servidores basado en Windows (WBMS).
- CVE-2023-21962 es un problema de seguridad al nivel de componentes de servicios de base de datos, lo que permite la accesión a datos vulnerables para las operaciones de sistema y de seguridad del servidor.
- CVE-2023-21955 es un problema de seguridad al nivel de cliente de servicio de base de datos, donde se puede acceder a datos críticos en una base de datos. 
- CVE-2022-21594 es un problema de seguridad del núcleo de Servicios de base de datos (dbsvcs), que permite el acceso a datos vulnerables desde la interfaz de cliente, lo que supone una amenaza para la seguridad de los servidores.
- Las vulnerabilidades se remedyan con la aplicación de patches de seguridad y el mantenimiento de las versiones de MySQL Server en desarrollo.
